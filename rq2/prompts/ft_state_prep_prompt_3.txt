Task: Task: Generate a fault tolerant version of the given quantum circuit, that preserves the given stabilizers.

Fault tolerance:
- A fault is a location in the circuit where there is an unexpected disruption (X, Y, Z Pauli gate) that alters the circuit's intended state in an undesired way.
- A circuit is fault tolerant if any fault propagates to less than floor(({distance} - 1)/ 2) data qubits
- A circuit is also fault tolerant if a fault that propagates to greater than or equal to floor(({distance} - 1)/2) data qubits triggers a flag ancilla that indicates a failure of the circuit. 

Flag gadget:
  A flag qubit f is an ancilla qubit added to a circuit to detect harmful error
  propagation. It is coupled with the data qubits so that it is triggered (measured with
  eigenvalue -1) if and only if a fault causes an error that propagates to more data qubits
  than the code can correct (i.e., beyond the correctable threshold t).
  You can find flag qubit strategies in https://arxiv.org/pdf/2408.11894

Setup:
  Let d = {distance} be the distance of the code, and let t be the correctable threshold:
    t = floor((d - 1) / 2)

  Let C be the circuit 
  ```
  {non_ft_circuit}
  ```
  . Let spots(C) be the set of valid fault locations in C where a single-qubit Pauli fault
  can occur. For each location l in spots(C) and P in {{X, Y, Z}}, let C[l <- P] denote the
  circuit obtained from C by inserting P at location l. Define the set of all single-fault
  variants:
    S(C) = {{ C[l <- P] : l in spots(C), P in {{X, Y, Z}} }}

  Let C' be the new fault tolerant version of C. Let E(C', C) be the number of errors in C' compared with C after propagation.

Fault-tolerance score:
  For C' in S(C), let pi(C') in {{0,1}} indicates whether any flag fires in C'. If no flag fires in circuit C', i.e., no flag qubit measured to be -1, then pi(C')=1. Otherwise, pi(C')=0. If a circuit C has flags that fires all the time without any insertion of any Pauli errors, set FT(C) to be invalid. 

  Let T(S(C)) denote the set of all circuits in S(C) whose error exceeds the
  correctable threshold t compared with C:
    T(S(C)) := {{ C' in S(C) | E(C', C) > t }}

  Then the fault-tolerance score FT(C) is defined as:
    - If T(S(C)) is non-empty, then: 
      FT(C) = 1 - (1 / |T(S(C))|) * sum over C' in S(C) of 1[pi(C') = 1 AND E(C', C) > t]           
    - If T(S(C)) is empty, then FT(C) = 1.
